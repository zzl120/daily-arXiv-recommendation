### [1] [SelRoute: Query-Type-Aware Routing for Long-Term Conversational Memory Retrieval](https://arxiv.org/abs/2604.02431)
*Matthew McKee*

Main category: cs.IR

TL;DR: SelRoute是一个用于长期对话记忆检索的选择性路由框架，通过基于查询类型将每个查询分配到专门的检索管道（词法、语义、混合或词汇增强）来优化检索性能。在LongMemEval_M基准测试中，SelRoute实现了Recall@5为0.800，优于现有最佳基线0.762，同时通过交叉验证和跨基准评估证明了其稳定性和泛化能力。


<details>
  <summary>Details</summary>
Motivation: 长期对话AI系统需要从跨越数月或数年的数百个对话会话中检索相关信息。现有方法主要依赖大型密集检索模型或LLM增强索引，但研究发现：1）没有单一检索方法在所有查询类型上表现最优；2）词法和语义检索存在互补性；3）查询类型是检索策略选择的强先验信息。这促使研究者开发一个能够根据查询类型动态选择最优检索策略的框架。

Method: SelRoute框架包含三个核心组件：1）基于SQLite FTS5的词法检索（BM25排名）；2）基于BERT类模型的语义检索（bge-small/bge-base等）；3）基于规则的词汇增强管道（超义词扩展、动作桥接、主题房间检测）。框架采用确定性路由机制，基于查询类型将查询分配到四种检索管道之一：纯词法检索、纯语义检索、混合检索（使用互惠排名融合）或增强词法检索。路由表通过经验分析从51个困难实例中推导得出，并通过5折分层交叉验证确认稳定性。

Result: 在LongMemEval_M基准测试中，SelRoute使用bge-base-en-v1.5模型实现了Recall@5为0.800，使用bge-small-en-v1.5模型实现了0.786，显著优于Contriever+事实键的0.762。零机器学习基线（仅使用SQLite FTS5）实现了NDCG@5为0.692，已超过所有已发布的基线。交叉验证显示泛化差距仅为1.3-2.4个百分点。在8个额外基准测试上的评估表明，相同的路由表无需特定调整即可泛化，但在推理密集型检索任务上存在明显失败模式。

Conclusion: 长期对话记忆检索是一系列具有不同检索特性的子问题。统一策略需要在查询类型之间妥协，而路由消除了这种妥协。关键发现是词汇增强有助于词法检索但会损害语义检索，这对混合系统设计具有实际意义。研究结果表明，当检索需求在可识别的查询类别之间系统性变化时，路由逻辑可能比模型规模带来更大的收益提升，但轻量级检索器之间的路由不能替代需要推理能力的任务。

Abstract: Retrieving relevant past interactions from long-term conversational memory typically relies on large dense retrieval models (110M-1.5B parameters) or LLM-augmented indexing. We introduce SelRoute, a framework that routes each query to a specialized retrieval pipeline -- lexical, semantic, hybrid, or vocabulary-enriched -- based on its query type. On LongMemEval_M (Wu et al., 2024), SelRoute achieves Recall@5 of 0.800 with bge-base-en-v1.5 (109M parameters) and 0.786 with bge-small-en-v1.5 (33M parameters), compared to 0.762 for Contriever with LLM-generated fact keys. A zero-ML baseline using SQLite FTS5 alone achieves NDCG@5 of 0.692, already exceeding all published baselines on ranking quality -- a gap we attribute partly to implementation differences in lexical retrieval. Five-fold stratified cross-validation confirms routing stability (CV gap of 1.3-2.4 Recall@5 points; routes stable for 4/6 query types across folds). A regex-based query-type classifier achieves 83% effective routing accuracy, and end-to-end retrieval with predicted types (Recall@5 = 0.689) still outperforms uniform baselines. Cross-benchmark evaluation on 8 additional benchmarks spanning 62,000+ instances -- including MSDialog, LoCoMo, QReCC, and PerLTQA -- confirms generalization without benchmark-specific tuning, while exposing a clear failure mode on reasoning-intensive retrieval (RECOR Recall@5 = 0.149) that bounds the claim. We also identify an enrichment-embedding asymmetry: vocabulary expansion at storage time improves lexical search but degrades embedding search, motivating per-pipeline enrichment decisions. The full system requires no GPU and no LLM inference at query time.

Motivation: 多模态推荐系统通过引入物品内容（如视觉和文本描述）来丰富物品表示，但其成功依赖于将这些内容模态与用户交互数据对齐。现有基于解耦表示学习的方法存在两个根本缺陷：1）假设物品内容对用户偏好的相关性对所有用户都是通用的，这与用户条件偏好的事实相矛盾；2）通过成对对比损失分别优化跨模态对齐，系统性地忽略了多个内容模态共同影响用户选择时固有的高阶依赖关系。

Method: GTC框架包含两个核心组件：1）用户感知生成过滤：使用交互引导的扩散模型执行用户感知的内容特征过滤，通过前向扩散过程向内容特征注入高斯噪声，然后通过以用户交互历史为条件的去噪网络反向生成仅保留与特定用户相关特征的内容表示；2）总相关优化：通过优化交互、视觉和文本表示之间总相关的可处理下界，捕获所有模态之间的完整依赖结构，包括成对方法忽略的高阶交互。最终通过相似性门控机制和残差连接融合对齐后的表示进行推荐。

Result: 在Sports、Baby和Cellphone三个标准多模态推荐基准数据集上的实验表明，GTC在所有指标上均显著优于现有最先进方法。在Sports数据集上，NDCG@5提升达28.30%，Recall@5提升25.67%，MAP@5提升29.69%。消融研究验证了条件偏好驱动特征过滤和总相关优化的有效性，证明了GTC在多模态推荐任务中建模用户条件关系的能力。

Conclusion: GTC通过用户条件生成过滤和总相关优化，克服了多模态推荐中普遍特征相关假设和不完整对齐的局限性，实现了更个性化和一致对齐的物品表示。该框架为未来多模态推荐研究开辟了新方向，强调了用户条件生成和整体对齐的重要性。

Abstract: Multi-modal recommendation (MMR) enriches item representations by introducing item content, e.g., visual and textual descriptions, to improve upon interaction-only recommenders. The success of MMR hinges on aligning these content modalities with user preferences derived from interaction data, yet dominant practices based on disentangling modality-invariant preference-driving signals from modality-specific preference-irrelevant noises are flawed. First, they assume a one-size-fits-all relevance of item content to user preferences for all users, which contradicts the user-conditional fact of preferences. Second, they optimize pairwise contrastive losses separately toward cross-modal alignment, systematically ignoring higher-order dependencies inherent when multiple content modalities jointly influence user choices. In this paper, we introduce GTC, a conditional Generative Total Correlation learning framework. We employ an interaction-guided diffusion model to perform user-aware content feature filtering, preserving only personalized features relevant to each individual user. Furthermore, to capture complete cross-modal dependencies, we optimize a tractable lower bound of the total correlation of item representations across all modalities. Experiments on standard MMR benchmarks show GTC consistently outperforms state-of-the-art, with gains of up to 28.30% in NDCG@5. Ablation studies validate both conditional preference-driven feature filtering and total correlation optimization, confirming the ability of GTC to model user-conditional relationships in MMR tasks.
