Most large scale ads recommendation systems use a cascading multi-stage paradigm with multi-routed candidate generation stage followed by one or more re-ranking stage . The key to improve the predictability of the system is to find the right semantic representation of the user and the items. For the scope of this paper, we focus on the candidate generation stage. The primary objective of a candidate generator is to increase the final stage recall, passing as many potentially relevant candidates to the next stage as allowed by infra capacity. Given that it is ultimately the semantic concepts that the users engage with, a good representation should be that can capture these high level concepts. A representation would be useful to facilitate a hierarchical, cluster based computational framework for an efficient processing. These two properties, semantic-aware and hierarchical, are thus the key properties we seek. And large-language models such as Llama is thus a good candidate model due to their: : LLMs process vast contextual information , enabling nuanced ad representations and more accurate candidate selection. : LLMs leverage ad inputs to capture semantic meaning, grouping similar ads closely in semantic space. : LLM generated metadata enables building a graph of ads, where nodes are ads and edges represent shared attributes, supporting efficient candidate expansion and system predictability. Our high level design to leverage LLM in ads candidate generation to improve system predictability is the following: We take a pre-trained LLM, fine-tune it on ads data set with retrieval specific tasks. In our case we already have an in-house fine-tuned LLM on ads engagement data sets. Use the LLM to infer hierarchical ad semantic attributes and a relationship graph based on the attribute similarity . Apply graph traversal algorithms to expand from seed ads, enabling semantic ad-to-ad similarity expansion and optimizing recall. The fundamental hypothesis is that the fine-tuned LLM would be able to better retrieve ad candidates that belong to the same semantic equivalence class that would otherwise been missed.
